# vue构建功能块

支持[vue-loader](vue-loader.vuejs.org)

## 安装

```shell
npm i jyb_jfet-build-block-vue --save
```

## 使用

更多用法见`jfet-build`的`开发构建功能块`[文档](../jfet-build/doc/DevelopBlock.md)


```javascript
const vue = require('jyb_jfet-build-block-vue');

// preset
createConfig(context, [
  vue(options)
]);
```
